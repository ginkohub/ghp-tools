Quick Integration

Add the hit counter to your project in seconds.

Best for GitHub READMEs or simple sites. Just paste the image URL.

![Visitors](https://ghp-tools.vercel.app/api/tools/hit-counter/YOUR_SITE_ID?format=badge&color=blue)

Best for accurate unique visitor tracking using localStorage.

<script>
  (async () => {
    let uid = localStorage.getItem('ghp_visitor_id') || crypto.randomUUID();
    localStorage.setItem('ghp_visitor_id', uid);
    
    const res = await fetch(`https://ghp-tools.vercel.app/api/tools/hit-counter/YOUR_SITE_ID?uid=${uid}`);
    const data = await res.json();
    console.log('Unique Visitors:', data.count);
  })();
</script>
